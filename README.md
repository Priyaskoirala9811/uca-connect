# UCA Connect

UCA Connect is a collaborative web application developed for students and tutors at the University for the Creative Arts (UCA). The platform was created as a Final Major Project to improve communication, collaboration, and project management between students across different campuses and courses.

## Main Features

- User authentication using Firebase
- Collaboration finder system
- Real-time group and direct messaging
- Shared task management board
- File upload support
- Notifications and updates
- Mobile responsive layout
- Live deployment using Vercel

## Technologies Used

- Next.js
- React
- TypeScript
- Firebase
- Firestore Database
- Firebase Authentication
- Firebase Storage
- Tailwind CSS
- GitHub
- Vercel
